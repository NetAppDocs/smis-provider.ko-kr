= 로깅 수준
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
기록되는 메시지 유형을 지정할 수 있습니다(예: 치명적인 시스템 메시지만 기록하려는 경우).

로깅 수준을 다음 중 하나로 구성할 수 있습니다.

* * TRACE * (트레이스 *)
+
cimserver_standard 로그에 추적 메시지를 저장합니다.

* 정보 *
+
모든 시스템 메시지(정보, 경고, 심각하고 치명적인 메시지)를 기록합니다.

* * 경고 *
+
경고, 심각하고 치명적인 시스템 메시지를 기록합니다.

* * 심각 *
+
심각하고 치명적인 시스템 메시지를 기록합니다

* * 치명적 *
+
치명적인 시스템 메시지만 기록합니다.
