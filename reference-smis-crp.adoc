= SMIS CRP
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
'Smis CRP' 명령은 NetApp SMI-S Provider 프로필을 포함하여 NetApp SMI-S Provider에서 지원하는 CIM 등록 프로필에 대해 설명합니다.

SMIS CRP

'[-t{ * http * | * https *}]'

'C:\Program Files (x86)\NetApp\SMIS\Pegasus\bin'을 선택합니다

유효한 사용자 이름과 암호를 가진 사용자

.매개 변수
* '*[-t{http|https}] *'
+
사용할 프로토콜: HTTPS(기본값) 또는 HTTP
