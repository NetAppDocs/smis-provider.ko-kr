= 연결이 거부되었습니다
:icons: font
:imagesdir: ./media/


* * 메시지 *
+
"연결이 거부되었습니다.

* * 원인 *
+
CIM 서버가 시작되지 않았습니다.

* * 시정 조치 *
+
NetApp SMI-S Provider를 설치한 디렉토리에서 "bin" 디렉토리로 이동한 후 다음 명령을 입력하여 CIM 서버가 시작되었는지 확인합니다.

+
'Mis cimserver status'입니다

+
CIM 서버가 실행되고 있지 않으면 다음 명령을 입력합니다.

+
'Mis cimserver start'입니다
