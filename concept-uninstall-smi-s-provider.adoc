= NetApp SMI-S Provider를 제거합니다
:icons: font
:imagesdir: ./media/


[role="lead"]
필요에 따라 SMI-S Provider를 제거할 수 있습니다. 예를 들어, 기존 설치 버전에 따라 최신 버전을 설치하기 전에 기존 SMI-S Provider 설치를 제거해야 할 수 있습니다.
